App.Pi <- function(n) {
    
    x = runif(n, 0, 1);
    y = runif(n, 0, 1);
    int = x ^ 2 + y ^ 2 < 1;
    
    # calcul de l'approximation de pi
    
    return (4 * (sum(int) / length(x1)));
}

Distribution.Pi <- function(n,p){

Exercice 2: Estimation d’une probabilité.¶

Commentaire : le mme type de simulation permet de donner des valeurs approximatives dune probabilit, voire de toutes les valeurs dune loi de probabilit On considre toujours notre carr carr [0,1]x[0,1], muni de ses quatre cots. On considre la variable alatoire X: distance au cot le plus proche.

  1. Donner une approximation de la probabilit que la distance du point au cot le plus proche soit infrieure d=0,25. Crer une fonction Cote(), qui prend comme argument n,d.

count <- function(n, d) {

    # cration des lois uniformes
    x <- runif(n, 0, 1)
    y <- runif(n, 0, 1)
    
    # calcul de la distance infrieur  1/4
    dist <- pmin(x, 1-x, y, 1-y) < d
    plot(x[dist], y[dist], col='green', pch=16, xlab='X', ylab='Y', main='Distance du points au ct le plus proche')
    points(x[!dist], y[!dist], col='red', pch=16)
    
    # calcul de la probabilit
    return (sum(dist)/length(x))
}
